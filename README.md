# WebGL-Renderer

This is a renderer for 3D models written in pure javascript using the WebGL API.

[Check out the Demo!](https://zmilla93.github.io/WebGL-Renderer)

## Features
- Phong Lighting Model (Ambient, Diffuse, Speuclar, and Emission)
- Point Lights
- Directional Lights
- Textures
- Material System
- .obj Models
